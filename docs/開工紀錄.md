# 開工紀錄

## 2026-06-23

本日正式進入開工狀態，先建立第一版可操作的靜態 MVP 原型。

## 本次開工範圍

- 建立 `web/index.html` 作為系統原型入口。
- 建立 `web/styles.css` 作為原型介面樣式。
- 建立 `web/app.js` 作為本機互動邏輯。
- 使用瀏覽器 `localStorage` 保存原型資料，不串接外部 API。
- 保持 API KEY 安全原則：本次沒有使用任何真實金鑰，後續串接第三方服務時需使用 `.env` 或部署平台 Secret。

## 第一版原型包含

- 今日看板
- 車主線上預約
- 時段容量顯示
- 廠內現場車流狀態更新
- 車輛資料與車齡判斷
- 下次檢驗提醒名單
- 時段容量設定

3. 確認第一版要先做純內部系統，還是同步開放車主端頁面。
4. 決定後端、資料庫與通知服務導入時機。
