# 車籍資料 Excel 匯入欄位設計

本專案提供 `data/車籍資料匯入範本.xlsx` 給驗車場整理既有車籍資料。廠家可將資料複製到範本欄位後上傳，系統會依車牌建立或更新車輛資料。

## 匯入原則

- 車牌號碼是主要識別欄位。
- 車牌空白的列不匯入。
- 選填欄位沒填不會影響資料載入。
- 若同一車牌重複匯入，原型會以最新匯入資料覆蓋先前 Excel 匯入資料。
- 正式版可再加入匯入前預覽、錯誤列下載、重複資料合併規則。

## 必填欄位

- 車牌號碼：車輛識別與搜尋用。
- 車主姓名：聯絡與資料顯示用。
- 手機：提醒通知與查詢用。

## 建議欄位

- 車種：預設可用自用小客車、自用小貨車、營業車、機車、其他。
- 出廠年月：用於判斷車齡與檢驗週期，格式建議 `YYYY-MM`。
- 上次檢驗日：用於推算下次提醒，格式建議 `YYYY-MM-DD`。
- 通知同意：建議填 `同意` 或 `未同意`。

## 選填欄位

- 初次領牌日
- 下次檢驗日
- 通知方式
- Email
- 地址
- 身分證末四碼
- 車身號碼末四碼
- 燃料種類
- 排氣量
- 備註

## 個資安全建議

- 不建議匯入完整身分證字號。
- 不建議匯入完整車身號碼，除非現場有明確管理需求。
- 若需要輔助核對，建議只保留末四碼。
- 匯入檔案與備份檔應由車廠內部妥善保存，不應放入公開資料夾。
- MVP 原型目前使用前端 Excel 解析示範流程；正式版建議將解析套件打包在系統內，避免車籍匯入頁依賴外部 CDN 腳本。
